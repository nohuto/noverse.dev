I'd rather use a modified version of Discord ([Vesktop](https://github.com/Vencord/Vesktop)/[Equibop](https://github.com/Equicord/Equibop)) to prevent tracking/telemetry, or if possible use discord via your browser.

## App Configuration (Equibop)

### Content & Social

- `Direct Message spam`: Do not filter

### Data & Privacy

- `Use data to improve Discord`: Off
- `Use my Discord activity to personalize Sponsored Content`: Off
- `use data to personalize my Discord experience`: Off
- `Allow my voice to be recorded in Clips`: Off
- `Use data to make Discord work`: Off

### Authorizied Apps

- Deauthorize all unnneded applications (don't authorizize unknown apps)

### Notifications

- `Communication Emails`: Off
- `Social Emails`: Off
- `Announcement and Update Emails`: Off
- `Tip Emails`: Off

### Plugins

- `ClearURLs`
- `NoTypingAnimation`
- `SilentTyping`
- `NoF1`
### Appearance (personal preference)

- `UI Density`: Compact
- `Chat Message Display`: Compact
- `Hide user avatars`: Off
- `Space Between Message Groups`: 0px
- `Chat font scaling`: 15px

### Accessibility

- `Dispaly Name Styles`: Off
- `Enable Reduced Motion`: On

### Activity Privacy

- `Share my activity`: Off
- `Automatically share my activity with`: Friends only
