I personally use [noai.duckduckgo](https://noai.duckduckgo.com/) (`https://noai.duckduckgo.com/search?q=%s`) everywhere. Other provider that seem to not collect your data are [Brave Search](https://search.brave.com/), [Startpage](https://www.startpage.com/), [Mojeek](https://www.mojeek.com/).

## DuckDuckGo Configuration

DuckDuckGo has several different versions of how their search engine works: [main](https://duckduckgo.com/), [lite](https://lite.duckduckgo.com) (non-JS), [html](https://html.duckduckgo.com) (non-JS), [noai](https://noai.duckduckgo.com/). I recommend to either use noai or html.

### [General](https://duckduckgo.com/settings#general)

- `Instant Answers`: Off
- `Hide AI-Generated Images`: On
- `Infinite Scroll for Images, Videos, and Shopping`: On
- `Infinite Scroll`: On
- `Autocomplete Suggestions`: Off
- `Open Links in a New Tab`: On
- `Advertisements`: Off
- `Location`: Off
- `Page Break Numbers`: Off
- `Page Break Lines`: Off
- `Install DuckDuckGo`: Off
- `Install DuckDuckGo reminders`: Off
- `Privacy Newsletters`: Off
- `Privacy Newsletter reminders`: Off
- `Homepage Privacy Tips`: Off
- `Help Improve DuckDuckGo`: Off

### [Appearance](https://duckduckgo.com/settings#appearance)

- `Font Size`: Medium (personal preference)
- `Compact Results`: On (personal preference)
- `'Protected' Reminder`: Off

### [Privacy](https://duckduckgo.com/settings#privacy)

- `GET Requests`: Off
- `Video Playback`: Prompt me
- `Redirect (When Necessary)`: On

### [AI Features](https://duckduckgo.com/settings#aifeatures) (doesn't exist if using noai.duckduckgo)

- `Duck.ai`: Off
- `Search Assist`: Never
- `Hide AI-Generated Images`: On

## [Brave](https://search.brave.com/settings) Search Configuration

- `Answer with AI`: Off
- `Open links in new tab`: On
- `Anonymous local results`: On
- `Anonymous usage metrics`: Off
