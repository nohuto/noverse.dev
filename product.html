Windows Configuration

WinConfig is based on my personal research on several topics, which I began documenting around July 2024.

Noteable Windows Configuration Tool Features:

  • Transparent execution logs so users can review every action, these include the value, data, type and if given the amount of modified keys. Specific options will show the PS code that got executed.
  • Dynamic state detection instead of relying on cached registry values, this includes static/dynamic (pattern) and all used cmdlets (which are also saved in the registry but on uncommon ways).
  • Each option is backed up by documentation, instead of just general "optimize" prompts with an useless text.
  • Every control is available individually instead of being merged into large bundles.
  • Supports dark/light modes, custom color schemes, and layout preferences for search filters, rows, columns, and panel widths.
  • Export/import your entire configuration within a single click, so you can easily reapply everything on new installations.

Data Privacy

WinConfig uses three hardware identifiers for license registration and verification, SMBIOS UUID, baseboard serial, and CPU ID (with MAC address or device IDs as fallbacks). Saved login data is stored locally, I don't have access to your password.

License

Each purchase grants one hardware bound license. Manual validation is required after checkout and can take several minutes or hours. Keys do not transfer across devices without re-registration.

After Purchase
