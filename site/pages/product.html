Windows Configuration

WinConfig is based on my personal research on several topics, which I began documenting around July 2024. It has transparent logs so users can see every change, including value name, data, type and if recursive the amount of modified keys, some may also show executed PS code. Each option has its own documentation, some may be more detailed whenever theres no public (trustworthy) documentation on them, some less, see below for details.

Terms of Service

Data Privacy

WinConfig uses three hardware identifiers for license registration and verification, SMBIOS UUID, baseboard serial, and CPU ID (with MAC address or device IDs as fallbacks). Saved login data is stored locally, I don't have access to your password.

Licenses are for personal use only. Reselling, transferring, redistributing, or SRE of WinConfig is prohibited unless you receive explicit written permission from me (Nohuto). By purchasing or using WinConfig, you accept these terms.

Refund Policy

Refunds are available only if no registration has been made and no Discord role has been assigned. Include proof of purchase when submitting a refund request.

License

Each purchase grants one hardware bound license. Manual validation is required after checkout and can take several minutes or hours. Keys do not transfer across devices without re-registration.

After Purchase

To get access, you must either include your Discord ID in the purchase message or send me your transaction ID so I can assign the Discord role. The role isn't assigned automatically, so one of these is required. Once the role is assigned, you'll get access to a download link for the tool. After downloading, you must register via the tool, and then I'll validate your registration. Joining (or already being in) the Discord server is recommended for now so you receive updates, announcements, and support. Otherwise you may miss app updates (config updates would theoretically work as long as no new section/helper gets added).
